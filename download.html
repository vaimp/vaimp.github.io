Requisitos

Antes de descargar VAIMP, asegúrate de que cumples con los siguientes requisitos:

  • Windows 8+ (32 o 64 bits)
  • .NET Framework 4.5 runtime (puedes descargarlo aquí)
  • Un extractor de archivos .rar para poder extraer la carpeta del programa (si no estás seguro de cuál instalar, se recomiendan WinRAR o 7-Zip)
  • Para poder controlar a VAIMP por voz, deberá funcionar el reconocimiento de voz de Windows en tu computadora, y deberá estar configurado en Inglés (Estados Unidos) o Español (España)
  • También deberás tener una carpeta que contenga todos tus archivos de música
  • Se recomienda tener acceso a internet para que así puedas descargar los metadatos de tus canciones y comprobar actualizaciones
