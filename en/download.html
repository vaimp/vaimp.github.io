Requirements

Before downloading VAIMP, ensure you meet the following requirements:

  • Windows 8+ (32 or 64 bits)
  • .NET Framework 4.5 runtime (you can download it here)
  • A .rar extractor to extract the program folder (if you are not sure about which one to install, I recommend you WinRAR or 7-Zip)
  • In order to control VAIMP by voice, Windows speech recognition must work on your computer, and it must be set to English (United States) or Spanish (Spain)
  • You must also have a folder that contains all your music files
  • It is recommended that you have access to the internet so that you can download the metadata of your songs and check for updates.
